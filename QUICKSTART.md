# Quick Start Guide

## Installation Steps

1. **Install Node.js dependencies:**
   ```bash
   npm install
   ```

2. **Start the development server:**
   ```bash
   npm start
   ```

3. **Run on your device:**
   - Install Expo Go app on your iOS/Android device
   - Scan the QR code displayed in the terminal
   - Or press `i` for iOS simulator / `a` for Android emulator

## First Time Setup

1. The app will show an onboarding screen on first launch
2. Create an account or use the demo login
3. Start tracking your mood and adding events!

## Key Features to Try

1. **Dashboard**: See your overview and quick stats
2. **Calendar**: 
   - Switch between Month/Week/Day views
   - Add events and wellness activities
   - View time blocks in Day view
3. **Mood Tracker**: Log your daily mood
4. **Notes**: Add quick notes and tasks
5. **Stats**: View your progress and insights
6. **Settings**: Customize colors and preferences

## Troubleshooting

- **Metro bundler issues**: Run `npm start -- --reset-cache`
- **Module not found**: Delete `node_modules` and run `npm install` again
- **Expo Go connection**: Make sure your device and computer are on the same WiFi network

## Notes

- All data is stored locally on your device
- The app works offline
- Assets (icons, splash screens) are optional - the app will work without them
